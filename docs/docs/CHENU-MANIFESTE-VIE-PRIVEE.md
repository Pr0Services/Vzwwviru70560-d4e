# CHE·NU — Manifeste de Vie Privée
### La dignité humaine avant la valeur des données

---

## Notre Promesse

CHE·NU est bâti sur une croyance simple:

**Les données personnelles ne sont pas du carburant.  
La vie privée n'est pas une fonctionnalité.  
La dignité humaine n'est pas négociable.**

---

## Ce que "Données Personnelles" signifie pour nous

Les données personnelles incluent tout ce qui peut:
- vous identifier
- vous décrire
- être relié à vous
- inférer quelque chose sur vous

Si ça touche une vie humaine — c'est protégé.

---

## Notre Règle Par Défaut

**Vos données sont privées par défaut.**

Pas de partage caché.  
Pas de collecte silencieuse.  
Pas de "vous avez accepté sans le savoir".

L'accès n'arrive que lorsque **vous le choisissez explicitement**.

---

## Pas de Surveillance, Pas de Profilage

CHE·NU ne fait **PAS**:
- suivre les émotions
- profiler les personnalités
- scorer la productivité
- inférer des traits psychologiques
- surveiller le comportement en arrière-plan

Nous refusons de transformer les humains en signaux.

---

## Vos Données Vous Appartiennent

Vous pouvez:
- lire vos données
- les exporter
- les supprimer
- les emporter avec vous

Quand les données quittent CHE·NU, elles perdent tout pouvoir système.
Elles sont **à vous**, pas à nous.

---

## Les Agents Servent — Ils Ne Surveillent Pas

Les agents IA dans CHE·NU:
- agissent uniquement dans un scope explicite
- ne peuvent pas accéder aux données personnelles silencieusement
- oublient ce qu'ils n'ont pas le droit de garder
- doivent déclarer quand des données sont touchées

Ils assistent.  
Ils n'observent pas en secret.

---

## Perspective Collective Sans Pression Individuelle

Quand CHE·NU observe des patterns à grande échelle:
- les individus ne sont jamais identifiables
- aucun groupe ne peut être ciblé
- aucune action ne peut être forcée

Conscience sans levier.

---

## La Loi Est le Plancher — Pas le Plafond

CHE·NU est conçu pour s'aligner avec:
- RGPD (Europe)
- CCPA / CPRA (Californie)
- PIPEDA (Canada)

Si la loi permet moins de protection que ce que nous promettons,
**nous choisissons le standard le plus élevé.**

---

## Notre Engagement Final

Si une fonctionnalité nécessite de violer la vie privée,
**cette fonctionnalité n'a pas sa place dans CHE·NU.**

La vie privée n'est pas un compromis.  
C'est une fondation.

---

*CHE·NU — Bâtir l'unité sans surveillance.*

❤️ Avec amour, pour l'humanité.
