# CHE·NU — Evolution Summary (Q1 2025)

## Context

This document summarizes the latest architectural and product decisions
to consolidate system direction and avoid divergence.

**Version:** 27.0.0  
**Last Updated:** December 14, 2025  
**Status:** Active Development

---

## Major Additions

### Governance Canon
- Immutable governance layer defined
- Clear authority hierarchy (User → Nova → Orchestrator → Agents → System)
- Explicit action flow enforced
- Tree Laws requiring human oversight

### Architecture Refactor
- **Surface → Zone** naming change
- 3 Zones: Interaction · Navigation · Conception
- Keyboard shortcuts: ⌘1 / ⌘2 / ⌘3
- Clean component structure

### Budget Architecture
- Unified budget system
- Presets + sphere + project + meeting
- Guards applied before execution
- Full token ledger implemented

### UX Foundations
- Pre-account minimal display
- Seamless transition to first login
- First 10 minutes UX defined
- Orchestrator activation rules clarified
- Expert Mode defined (⌘K command palette)

### Nova Evolution
- Nova defined as governance layer (not assistant)
- Voice activation rules established
- Avatar activation rules established
- Nova never executes actions
- Nova explains, proposes, waits for confirmation

### Meetings
- Meetings defined as governed workspaces
- AI assistance optional and scoped
- No default recording or transcription
- Budgets enforced per meeting

### XR / Spatial Extension
- XR defined as interface-only layer
- No governance changes
- Same confirmation and budget rules
