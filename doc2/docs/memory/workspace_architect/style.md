```json
{
  "workspace": {
    "id": "WS_001",
    "name": "Workspace Name",
    "grid": {"columns": 12, "rows": "auto"},
```

## Naming

- Workspace: `WS_[NAME]_[VERSION]`
- Panel: `PANEL_[TYPE]_[NUM]`
- Zone: `ZONE_[POSITION]`
