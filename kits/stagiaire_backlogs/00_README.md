# CHE-NU — Kit Stagiaire & Backlogs (v1.0)
Date: 2026-01-07

Ce zip contient les **spécifications canoniques** de l’Agent Stagiaire (apprentissage qualitatif) + un **guide d’implémentation** (sans imposer de stack).

## Contenu
- `01_CHARTE_AGENT_STAGIAIRE.md` — Charte et rôle
- `02_DEFINITION_BONNE_QUESTION.md` — Ce qu’est une “bonne question”
- `03_LISTE_INFOS_A_COLLECTER.md` — Données minimales + ce que le stagiaire refuse
- `04_CHARTE_SILENCE_ABSOLU.md` — Règles de silence
- `05_PROMOTION_PAR_SPHERES.md` — Quand une note mérite promotion (sphères/secteurs)
- `06_ACTIVATION_FIN_CONVERSATION_15MIN.md` — Activation “fin de conversation” + cooldown 15 min
- `07_GUIDE_IMPLEMENTATION.md` — Comment l’implémenter (états, événements, stockage, UI)
- `schemas/` — Schémas JSON
- `examples/` — Exemples de payloads

## Idée maîtresse
Le stagiaire n’est **pas** un agent qui répond.
C’est un agent qui **filtre le bruit à la source** et protège la qualité de l’apprentissage futur.
