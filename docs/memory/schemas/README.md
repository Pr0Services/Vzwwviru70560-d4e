# CHE·NU SCHEMAS

This folder contains global JSON schemas for all CHE·NU data structures.

## Available Schemas

| Schema | Purpose |
|--------|---------|
| `XR_SCENE.schema.json` | XR scene definitions |
| `UC_MAP.schema.json` | Cartography map structures |
| `HF_SCENE.schema.json` | HyperFabric topology |
| `WORKSPACE.schema.json` | Workspace layouts |
| `MORPHOTYPE.schema.json` | Symbol definitions |
| `PROJECTION.schema.json` | Projection configs |
| `DEPTH.schema.json` | Depth layer configs |

## Usage

All role outputs should validate against these schemas.
