# 03 — Le Fichier de Recentrage

## Fonction
Permettre au système de **revenir à une compréhension déjà stabilisée** quand il dérive.

## Contenu autorisé
- Intentions validées
- Décisions explicites
- Autorisations déjà données
- Structures qui ont fonctionné
- Hypothèses clôturées

## Contenu interdit
- Logs bruts
- Conversations complètes
- Hypothèses ouvertes
- Doutes non résolus

## Usage
- Utilisé uniquement par l’orchestreur
- Jamais exposé tel quel à l’utilisateur
- Sert à stopper la dérive, pas à expliquer
