# CHE·NU — VISUAL THEMES (Block 1)

> *"Same system. Same laws. Same agents. Only visual language changes."*

---

## CORE PRINCIPLE

Themes are **purely perceptual skins**.

They must **NEVER** modify:
- ❌ Logic
- ❌ Authority
- ❌ Agents
- ❌ Timelines
- ❌ Rules

---

## GLOBAL CONSTRAINTS

### Themes NEVER:
- Accelerate decisions
- Hide rules
- Change authority
- Alter agent behavior

### Theme Switching:
- Manual only
- Smooth transition
- No session reset

---

## THEME 1 — REALISTIC / PROFESSIONAL

### Intent
**Neutral, reassuring, serious, predictable.**

This is the **default** and **foundation** theme.

### Use Cases
- Daily use
- Institutions
- Investors
- Debug / control
- Long sessions

### Visual Rules

| Element | Rule |
|---------|------|
| Geometry | Simple rectangles, soft corners, strict grids |
| Colors | Off-white, light grey, dark grey, ONE blue accent |
| Typography | Clean sans-serif, strict hierarchy |
| Effects | No decorative effects |
| Emotion | No emotional cues |

### Agents
- Abstract icons (circle or square)
- Neutral colors only
- Text > visual presence

### Guards
- Subtle lock icon
- Neutral message
- No red, no alarm tone

### XR Environment
- Clean control room
- Flat floating panels
- Diffuse lighting
- No particles

### Goal Feeling
> *"Everything is clear and under control."*

---

## THEME 2 — AVATAR / HUMAN-CENTERED

### Intent
**Presence and collaboration without loss of human authority.**

### Use Cases
- Meeting rooms
- Coaching
- Collaboration
- Guided sessions

### Visual Rules

| Element | Rule |
|---------|------|
| Geometry | Softer rounded shapes |
| Colors | Warm but muted |
| Spacing | Generous |
| Typography | Same as Theme 1 |

### Avatars
- Stylized, non-realistic
- Limited calm expressions
- No dominance posture
- Respect personal space

### Agents
- Appear as subtle presences
- Never visually dominant

### XR Environment
- Circular space
- Respectful interpersonal distance
- Slow gestures
- Neutral gaze

### Interaction
- Clear turn-taking
- Agents may stay silent
- No forced engagement

### Goal Feeling
> *"I'm accompanied, not controlled."*

---

## THEME 3 — ANCIENT / BUILDER CIVILIZATION

### Intent
**Weight, continuity, responsibility through time.**

### Use Cases
- Methodology
- Governance
- Long-term vision
- Institutions

### Visual Rules

| Element | Rule |
|---------|------|
| Geometry | Massive, simple |
| Materials | Stone / wood |
| Lighting | Natural light |
| Symbols | Minimal |

### Timeline
- Visually engraved
- Decisions appear as inscriptions
- Rollback = reading position shift only

### Agents
- Represented as tablets, markers, or steles
- Non-anthropomorphic
- No idle animation

### XR Environment
- Temple / monolith / tree structure
- Natural light source
- Slow, grounded ambiance

### Guards
- Feel like natural laws
- Never punitive

### Goal Feeling
> *"What happens here carries responsibility."*

---

## THEME COMPARISON MATRIX

| Aspect | Realistic | Avatar | Ancient |
|--------|-----------|--------|---------|
| **Geometry** | Rectangles, grids | Rounded, soft | Massive, simple |
| **Colors** | Grey + blue | Warm, muted | Earth tones |
| **Agents** | Abstract icons | Subtle presences | Tablets/steles |
| **XR** | Control room | Circular space | Temple |
| **Feeling** | Control | Accompaniment | Responsibility |
| **Animation** | Minimal | Slow gestures | None/grounded |
