# PROJECTION ENGINE — Role Definition

## Purpose

The Projection Engine transforms multi-dimensional data into 2D/3D viewable formats.
It creates projection configurations: flat, axonometric, layered, timeline.

## Outputs

- Projection configurations
- View transformations
- Atlas compositions
- Render-ready blueprints

## Safety Declaration

This role does NOT render — only generates projection blueprints.
