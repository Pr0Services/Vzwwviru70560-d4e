## Security & Privacy

### Your Data

**What CHEÂ·NU knows:**
- Your profile information
- Threads you create
- Messages you send
- Files you upload

**What CHEÂ·NU doesn't know:**
- Your passwords (hashed with bcrypt)
- Payment details (handled by Stripe)
- Private external data (unless you share)

**Data encryption:**
- âœ… In transit: TLS 1.3
- âœ… At rest: AES-256
- âœ… Backups: Encrypted
- âœ… Logs: Anonymized

### MFA (Multi-Factor Authentication)

**Why use MFA?**
- Prevents 99.9% of account takeovers
- Protects even if password is compromised
- Required for enterprise features

**How it works:**
1. You login with email + password
2. System asks for 6-digit code
3. You open authenticator app
4. Enter the code
5. Access granted

**Backup codes:**
- You get 10 one-time codes
- Use if you lose your phone
- Each code works once
- Generate new codes after use

**Lost your phone?**
1. Click "Use backup code" at login
2. Enter one of your saved codes
3. Login granted
4. **Immediately**:
   - Disable old MFA
   - Set up new MFA
   - Generate new backup codes

### Privacy Settings

**Who can see your data:**

**Default (Private):**
- Only you see your threads
- Only you see your files
- Only you see your agents

**When shared:**
- You explicitly share a thread
- You add someone to a project
- You grant team access

**Control sharing:**
- Settings â†’ Privacy
  - Default sharing: Private
  - Allow team requests: Yes/No
  - Public profile: Yes/No
