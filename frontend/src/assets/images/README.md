# CHE·NU Logo Assets

## Logo Principal

Placer le fichier logo ici:
- `chenu-logo.jpg` ou `chenu-logo.png`

Le logo CHE·NU représente une pyramide maya avec:
- Végétation tropicale luxuriante
- Fontaine/bassin d'eau turquoise
- Texte "CHE NU" en lettres dorées
- Cercle englobant

## Dimensions recommandées

- **Original**: 1024x1024 px minimum
- **Favicon**: 32x32 px et 16x16 px
- **Icon**: 192x192 px (PWA)

## Chemins utilisés dans l'application

```
/public/chenu-logo.jpg
/public/assets/images/chenu-logo.jpg
/src/assets/images/chenu-logo.jpg
```

## Fallback

Si l'image n'est pas trouvée, un SVG stylisé est affiché automatiquement.
