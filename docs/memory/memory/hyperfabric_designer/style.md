# HYPERFABRIC DESIGNER — Output Style Guide

## HF_NODE Structure

```json
{
  "hf_node": {
    "id": "HF_NODE_001",
    "coordinates": {
      "x": 0, "y": 0, "z": 0,
      "t": "2025-01-01",
      "s": "concept_name",
      "p": 1.0
    },
    "type": "data|concept|event",
    "safe": true
  }
}
```

## Naming

- Node: `HF_NODE_[TYPE]_[NUM]`
- Link: `HF_LINK_[FROM]_[TO]`
- Slice: `HF_SLICE_[AXIS]_[VALUE]`
